# 🛍️ AuraShop

A modern Flutter-based E-Commerce application built with Firebase Authentication and Cloud Firestore. AuraShop provides a clean, responsive, and user-friendly shopping experience with secure authentication and real-time product management.

---

## ✨ Features

- 🔐 User Login & Signup using Firebase Authentication
- 🛒 Shopping Cart
- 📦 Product Listing
- 🔍 Search Products
- ❤️ Wishlist
- 👤 User Profile
- ☁️ Cloud Firestore Database
- 📱 Responsive UI
- ⚡ Fast and Smooth Performance

---

## 🛠️ Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Provider
- Shared Preferences
- Google Fonts

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Azanshahzad/AuraShop.git
```

### Navigate to the project

```bash
cd AuraShop
```

### Install dependencies

```bash
flutter pub get
```

### Run the app

```bash
flutter run
```

---

## 📌 Requirements

- Flutter SDK
- Android Studio or VS Code
- Firebase Project
- Android Emulator or Physical Device
